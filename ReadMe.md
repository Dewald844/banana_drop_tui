# Banana catch

---

![Demo image](image.jpeg)

---

#### Prerequisites
- Rust compiler service

---

#### Setup
- clone this repo
- inside cloned repo ```bash cd banana_drop_tui```
- ```bash cargo build```
- ```bash cargo run```


---

### Controls

- <- -> Arrows keys to move left and right
- Q to quit the game
- R to Restart after death

---

### Power ups
- "+" Add extra live
- "=" Extends bowl width
- "-" Decreses bowl width
